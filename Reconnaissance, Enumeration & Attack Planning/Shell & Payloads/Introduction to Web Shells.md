## What is a Web Shell?
- ده عباره عن browser-based shell session ممكن نستخدمه عشان نتفاعل مع الos بتاع الweb server
- طيب عشان نقدر ناخد web shell علي موقع معين لازم يكون الموقع ده مصاب بثغره معينه واحنا نستغلها وخصوصا لو الثغره دي كانت file upload
- معظم الpayloads اللي بتؤدي ل RCE بتكون شغاله بfunctionality رفع الملفات بحيث هحط كود ضار ف الpayload يتنفذ
